## Ch02 Java 기초 프로그래밍

#### -클래스(class)
: Java가 속한 객체 지향 언어의 기초적인 빌딩 블록이다. 하나의 자바 프로그램은 클래스들의 집합이다.

#### -메소드(method)
: 특정한 작업을 수행하는 코드의 묶음

#### -문장(statement)
: 사용자가 컴퓨터에게 작업을 지시하는 단위

#### -변수
: 데이터를 담아두는 상자로 비유(자세한 설명 예정)

#### -자료형
기초형(primitive type)
: 정수형, 실수형, 문자형, 논리형으로 분류 가능

참조형(reference type)
: 클래스, 배열, 인터페이스

#### -System.in과 Scanner 객체
: 이를 통해 콘솔에서 정수나 실수, 문자열을 리딩

#### -import 문장
: 다른 클래스를 포함시키는 문장

#### -산술연산자
: +, -, *, /, %

#### -관계연산자
: <, >, <=, >=

#### -논리연산자
: &&, ||, !
